# AI Survey Call Agent

Automated phone survey system using Twilio and ElevenLabs.

## Features
- Automated outbound calling
- AI-powered voice responses
- Survey management
- Real-time campaign tracking
- Contact management

## Deployment
This app is configured for Heroku deployment with PostgreSQL.

## Environment Variables
- TWILIO_ACCOUNT_SID
- TWILIO_AUTH_TOKEN  
- TWILIO_PHONE_NUMBER
- ELEVENLABS_API_KEY
- BASE_URL
- DATABASE_URL (auto-set by Heroku)
